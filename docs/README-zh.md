# 文档目录

本目录包含项目文档，记录了项目开发过程中的重要变更、解决方案和配置说明。

## 文档列表

### 项目结构变更
- **spear-next-library-conversion-zh.md** - spear-next 项目转换为纯库项目的详细记录
- **spear-next-library-conversion-en.md** - spear-next Project Conversion to Pure Library (English)

## 文档说明

### 命名规范
- 中文文档以 `-zh.md` 结尾
- 英文文档以 `-en.md` 结尾
- 文档名称使用小写字母和连字符分隔

### 内容范围
本目录记录以下类型的文档：
1. **项目结构变更** - 重要的代码结构调整和重构
2. **配置修改** - 构建配置、依赖管理等变更
3. **问题解决方案** - 开发过程中遇到的问题及解决方法
4. **最佳实践** - 项目开发中总结的经验和规范

### 使用目的
- 为未来的开发者提供项目历史和变更记录
- 便于其他 AI 工具理解项目结构和设计决策
- 作为项目知识传递的重要载体
